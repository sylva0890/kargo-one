# kargo-one
Kargo Hackathon Repository

Get depedency
mix deps.get

Create database
mix ecto.create

Migrate models
mix ecto.migrate

Run seeder
mix run priv/repo/seeds.exs

Serve application
mix phx.server

First run (get depedency, create database, migrate model, run seeder)
mix setup

First database, model and seeds setup
mix ecto.setup

Reinitialize database, model and seeds
mix ecto.reset


**Frontend**
#Install depedency
- run npm i

#Run App
- run npm start
